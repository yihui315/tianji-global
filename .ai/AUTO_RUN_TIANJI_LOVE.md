# AUTO RUN: TianJi Love

## Command

When user says:

```text
自动运行 TianJi Love
```

Codex must execute:

```text
TianJi Love Autonomous Money-Making Automation Controller
```

## Default behavior

Run safe autonomous loop:

```text
1. Read current gate.
2. Read KPI.
3. Generate daily publishing pack if missing.
4. Analyze KPI if data exists.
5. Improve content calendar.
6. Prepare source improvements only when safe.
7. Run validations.
8. Commit and push safe changes.
9. Update `.ai/CHANGELOG_AI.md`.
10. Update `.ai/REVIEW_PACKET.md`.
```

## Stop conditions

Stop and request approval if task requires:

```text
Stripe checkout
paid smoke
production deploy
server mutation
.env access
Supabase mutation
provider live AI call
webhook replay
social account credential use
```

## Required approval phrases

```text
批准跑 Stripe test-mode paid smoke
批准 production canary
批准服务器 staging 操作
批准真实发布到社交平台
```

## Never do automatically

```text
Never run live payment.
Never deploy production.
Never read or print secrets.
Never publish to social platforms using credentials without approval.
Never fake user results/testimonials.
```
