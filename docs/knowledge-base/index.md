# TianJi 知识库

> 本知识库涵盖八字、紫微斗数、七政四余、西方占星四大命理系统的核心概念。
> 所有内容经过文化一致性检查，确保各系统概念不混用。

## 命理系统

### [八字命理](./bazi/overview)
四柱八字，以天干地支配合五行十神，解读人生运势。

**核心概念：** 用神、格局、大运流年、十神
**入门文章：** [五行理论](./bazi/five-elements.md)

### [紫微斗数](./ziwei/overview)
十四主星配十二宫位，以星曜组合解读命运。

**核心概念：** 主星、四化星、十二宫位
**入门文章：** [十四主星详解](./ziwei/main-stars.md)

### 七政四余 ⏳筹备中
以日、月、五星及罗睺、计都、紫气、月孛论命。

### 西方占星 ⏳筹备中
以太阳星座为核心，配合行星相位和宫位。
