# 八字命理概述

## 什么是八字？

八字，又称四柱八字，是中国传统命理学的核心体系。以出生年、月、日、时四柱，配合天干地支，解读人生运势。

## 四柱结构

| 柱 | 含义 | 时间单位 |
|----|------|----------|
| 年柱 | 祖辈/童年/根基 | 60年一循环 |
| 月柱 | 父母/青年/事业 | 12年一循环 |
| 日柱 | 自己/中年/婚姻 | 60年一循环 |
| 时柱 | 子女/晚年/归宿 | 12年一循环 |

## 天干地支

十天干：甲乙丙丁戊己庚辛壬癸
十二地支：子丑寅卯辰巳午未申酉戌亥

## 五行与十神

五行：木火土金水，相生相克
十神：比肩、劫财、食神、伤官、正财、偏财、正官、七杀、正印、偏印

## 八字的核心概念

### 用神
用神是八字中的关键平衡点。根据五行旺衰，判断喜用之神，忌神之方位。

### 格局
格局决定人生主线：正官格、七杀格、正财格、偏财格、食神格、伤官格等。

### 大运流年
大运：每10年一个运势周期
流年：每一年的具体运势
流月：每个月的详细运势

## 与其他系统的区别

八字以立春为新年，不是农历正月初一。
八字以出生时辰为"时柱"，2小时为一个时辰。

## 适用场景

- 事业财运分析
- 婚恋配对参考
- 人生阶段规划
- 个性天赋解读
