# 五行理论

## 五行概述

木火土金水，五行循环相生，隔位相克。

## 相生关系
木生火 → 火生土 → 土生金 → 金生水 → 水生木

## 相克关系
木克土 → 土克水 → 水克火 → 火克金 → 金克木

## 五行与人

| 五行 | 方位 | 颜色 | 季节 | 脏腑 |
|------|------|------|------|------|
| 木 | 东 | 青绿 | 春 | 肝胆 |
| 火 | 南 | 红 | 夏 | 心小肠 |
| 土 | 中 | 黄 | 四季 | 脾胃 |
| 金 | 西 | 白 | 秋 | 肺大肠 |
| 水 | 北 | 黑 | 冬 | 肾膀胱 |

## 旺相休囚死

五行在四季中的旺衰状态：
- 旺：当令而旺
- 相：次旺，被月令所生
- 休：退休，被旺者生
- 囚：困囚，被旺者克
- 死：死绝，被旺者所克

## 五行与性格

木：仁慈、正直、生长
火：热情、冲动、礼仪
土：诚信、厚重、稳定
金：义气、决断、肃杀
水：智慧、灵活、变通
