# 十四主星详解

## 北斗六星

### 紫微星
- 五行：土
- 比喻：帝王、首领
- 特性：尊贵、权威、孤独
- 优点：领导力强、志向高远
- 缺点：主观强、孤僻

### 天机星
- 五行：乙木
- 比喻：谋士、智者
- 特性：机智、善变、多思
- 优点：头脑灵活、善于策划
- 缺点：犹豫不决、计较

### 太阳星
- 五行：丙火
- 比喻：光明、正义
- 特性：热情、正直、无私
- 优点：乐于助人、名声好
- 缺点：冲动、爱面子

### 武曲星
- 五行：辛金
- 比喻：将军、财务
- 特性：刚毅、果决、重义
- 优点：执行力强、理财好
- 缺点：固执、寡情

### 天同星
- 五行：壬水
- 比喻：福星、温和
- 特性：温和、享乐、知足
- 优点：人缘好、乐观
- 缺点：懒散、逃避

### 廉贞星
- 五行：丁火
- 比喻：次桃花、复杂
- 特性：感情丰富、复杂
- 优点：多才多艺、魅力
- 缺点：感情纠纷、固执

## 南斗五星

### 天府星
- 五行：戊土
- 比喻：皇后、财库
- 特性：稳重、保守、理财
- 优点：管理能力强、储蓄
- 缺点：保守、计较

### 天梁星
- 五行：戊土
- 比喻：长辈、荫蔽
- 特性：清高、济人、周期
- 优点：逢凶化吉、成熟
- 缺点：倚老卖老、唠叨

### 天相星
- 五行：壬水
- 比喻：宰相、辅佐
- 特性：公正、服务、跟随
- 优点：协调能力强、公正
- 缺点：依赖、无主见

### 七杀星
- 五行：庚金
- 比喻：先锋、刚猛
- 特性：冲动、决断、变化
- 优点：开拓力强、勇敢
- 缺点：鲁莽、暴烈

### 破军星
- 五行：癸水
- 比喻：破坏、创新
- 特性：破坏、消耗、创新
- 优点：突破常规、改革
- 缺点：破坏性强、不持久

## 中天星系

### 太阴星
- 五行：癸水
- 比喻：月亮、阴性
- 特性：温柔、内敛、暗藏
- 优点：细腻、深沉、包容
- 缺点：敏感、优柔、消极

### 贪狼星
- 五行：甲木
- 比喻：贪婪、桃花
- 特性：欲望、占有、多元
- 优点：多才多艺、适应强
- 缺点：贪心、花心、多变

### 巨门星
- 五行：癸水
- 比喻：是非、口舌
- 特性：分析、怀疑、口才
- 优点：洞察力强、能言
- 缺点：挑剔、搬弄是非
