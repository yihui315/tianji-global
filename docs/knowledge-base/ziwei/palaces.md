# 十二宫位详解

## 宫位概述

紫微斗数的十二宫位是解读人生各个方面的核心框架。与八字不同，紫微斗数以宫位为核心，结合星曜组合来分析命运。

### 十二宫位对应人生面向

| 宫位 | 人生面向 | 主要含义 |
|------|----------|----------|
| 命宫 | 本身 | 性格、命运核心、外貌气质 |
| 兄弟宫 | 手足 | 兄弟姐妹关系、朋友、同事 |
| 夫妻宫 | 婚姻 | 配偶、婚姻、感情 |
| 子女宫 | 子息 | 子女缘分、桃花、创作 |
| 财帛宫 | 财富 | 财运、理财方式、赚钱能力 |
| 疾厄宫 | 健康 | 身体状况、疾病、意外 |
| 迁移宫 | 变动 | 外出发展、旅行、贵人 |
| 仆役宫 | 人际 | 下属、合作伙伴、人际网络 |
| 官禄宫 | 事业 | 事业、地位、仕途 |
| 田宅宫 | 家宅 | 房产、祖业、家庭环境 |
| 福德宫 | 福分 | 享受、品德、福报 |
| 父母宫 | 长辈 | 父母、师长、学历 |

## 命宫

命宫是整个命盘的核心，代表命主本人。

### 命宫的重要性

命宫强则人生有主心骨，命宫弱则人生缺乏方向感。命宫也是判断外貌和性格的主要宫位。

### 命宫的核心要素

1. 命宫主星：决定基本性格
2. 命宫三方：官禄宫、财帛宫、迁移宫
3. 命宫四化：显示命运主线

### 命宫星曜组合

**紫微星在命宫**：尊贵、有领导力、孤独感

**天机星在命宫**：聪明、善变、策划能力强

**太阳星在命宫**：热情、正直、爱面子、视力问题

**武曲星在命宫**：刚毅、果决、寡言、财运佳

**天同星在命宫**：温和、懒散、享乐、人缘好

**廉贞星在命宫**：感情丰富、复杂、桃花

### 命宫空的含义

命宫无主星时，需看三方四正配合。命宫空不一定命薄，要综合判断。

## 兄弟宫

兄弟宫反映兄弟姐妹关系，也代表朋友和同事关系。

### 兄弟宫的基本含义

- 兄弟姐妹的人数和关系
- 平辈朋友的数量和质量
- 同事之间的关系
- 财务上的合作者

### 兄弟宫星曜分析

**紫微星**：朋友地位高，助力大

**天机星**：朋友变动大，人际网络广

**太阳星**：朋友热情，乐于助人

**武曲星**：朋友寡情，合作需谨慎

**天同星**：朋友和善，人缘好

### 兄弟宫与合伙

合伙创业需重点看兄弟宫：
- 兄弟宫吉星多，合作顺利
- 兄弟宫煞星多，合作易生变故
- 兄弟宫化忌，防财务纠纷

## 夫妻宫

夫妻宫是紫微斗数中最重要的宫位之一，直接关系到婚姻质量。

### 夫妻宫的核心要素

1. 配偶的性格和能力
2. 婚姻早晚
3. 婚姻质量
4. 婚后运势变化

### 夫妻宫星曜组合

**紫微星**：配偶尊贵、有地位、占有欲强

**天机星**：配偶聪明、变动多、沟通好

**太阳星**：配偶热情、正义、爱表现

**武曲星**：配偶刚毅、寡言、理财好

**天同星**：配偶温和、懒散、享受型

**太阴星**：配偶温柔、内敛、感情细腻

**贪狼星**：配偶桃花重、欲望强、复杂

**廉贞星**：配偶感情丰富、复杂、桃花

### 理想夫妻宫组合

- 紫微天府同宫：配偶有地位，婚姻稳定
- 太阳太阴并照：阴阳调和，婚姻和谐
- 天同巨门组合：感情深厚但需防口舌

### 需注意的夫妻宫组合

- 廉贞贪狼同宫：感情复杂，易生变故
- 紫微破军同宫：婚姻波动大
- 夫妻宫化忌：感情困扰多

## 子女宫

子女宫不仅代表子女缘分，也代表桃花和创作能力。

### 子女宫的多重含义

1. 子女缘分：有儿还是有女
2. 桃花运势：感情的机缘
3. 创作能力：艺术创作的天赋
4. 娱乐活动：休闲生活

### 子女宫星曜分析

**紫微星**：子女尊贵，有成就

**天机星**：子女聪明，多变动

**太阳星**：子女热情，有担当

**武曲星**：子女刚毅，财运好

**天同星**：子女温和，人缘好

**贪狼星**：桃花重，子女欲望强

### 子女宫与桃花

子女宫为桃花位：
- 贪狼星在子女宫：桃花强
- 廉贞星在子女宫：感情复杂
- 文昌文曲在子女宫：才艺出众

## 财帛宫

财帛宫直接反映命主的财运和理财能力。

### 财帛宫的基本含义

1. 赚钱能力和方式
2. 财运好坏
3. 理财风格
4. 财务状况

### 财帛宫星曜组合

**紫微星**：财运佳，有理财观念

**天机星**：财运变动，智慧生财

**太阳星**：财运起伏，理财忙碌

**武曲星**：财运稳定，擅长理财

**太阴星**：财运暗藏，女命佳

**贪狼星**：财运波动，欲望强

### 财帛宫四化分析

- 化禄：财运亨通，财源广进
- 化权：理财能力强，有掌财能力
- 化科：名声带来财运，理财有道
- 化忌：财务问题，破财或纠纷

## 疾厄宫

疾厄宫反映身体健康状况和潜在疾病。

### 疾厄宫的重要性

疾厄宫虽不如其他宫位显眼，但关系到人生的基本质量。需要提前预防。

### 各星曜与疾病

**紫微星**：脾胃问题，帝王病

**天机星**：肝胆问题，神经衰弱

**太阳星**：眼疾，高血压，心脏问题

**武曲星**：呼吸系统，肺部问题

**天同星**：泌尿系统，水肿

**太阴星**：妇科疾病，血液问题

**贪狼星**：肝胆，桃花相关的疾病

**巨门星**：口舌是非，消化系统

### 疾厄宫煞星影响

- 擎羊陀罗：外伤，手术
- 火星铃星：急性炎症
- 地劫地空：慢性消耗

## 迁移宫

迁移宫代表外出发展、旅途、贵人运。

### 迁移宫的核心含义

1. 外出发展的机遇
2. 旅途安全
3. 贵人运
4. 社会地位

### 迁移宫星曜分析

**紫微星**：外出有地位，宜往外发展

**天机星**：变动多，机会与挑战并存

**太阳星**：往光明方向发展，宜公开事业

**武曲星**：往武职或财务方向发展

**天同星**：往稳定方向发展，享受型

### 迁移宫与贵人

迁移宫有吉星：外出易遇贵人

迁移宫有煞星：外出易有小人

## 仆役宫

仆役宫代表下属、合作伙伴，也反映人际网络的广度。

### 仆役宫的基本含义

1. 下属的数量和能力
2. 合作伙伴的质量
3. 人际关系的广度
4. 团队运

### 仆役宫星曜组合

- 吉星多：下属得力，合作顺利
- 煞星多：下属难管，合作生变
- 化忌：防下属背叛或合作纠纷

## 官禄宫

官禄宫直接关系到事业发展和仕途。

### 官禄宫的核心要素

1. 事业发展方向
2. 社会地位
3. 管理能力
4. 学业学历

### 官禄宫星曜组合

**紫微星**：管理岗位，有领导力

**天机星**：策划类工作，变动行业

**太阳星**：公开事业，销售，教育

**武曲星**：财务金融，技术类

**天同星**：服务行业，悠闲职业

**太阴星**：幕后工作，艺术类

### 官禄宫四化分析

- 化禄：事业顺利，财运随来
- 化权：创业发展，有管理能力
- 化科：学术发展，名声提升
- 化忌：事业阻碍，变动多

## 田宅宫

田宅宫关系到房产、家庭和祖业。

### 田宅宫的多重含义

1. 房产运：置产时机和类型
2. 家庭环境：居住条件
3. 祖业：继承的产业
4. 家庭关系：家人相处

### 田宅宫星曜分析

**紫微星**：田宅多，宜置产

**天机星**：房产变动，不动产交易

**太阳星**：明亮宅府，讲究风水

**武曲星**：财务型房产，投资回报

**太阴星**：阴宅，房产运强（尤其女命）

### 田宅宫煞星影响

- 火星铃星：房产纠纷
- 地劫地空：房产消耗
- 擎羊陀罗：产权问题

## 福德宫

福德宫代表命主的福报、品德和享受。

### 福德宫的基本含义

1. 先天福报
2. 品德修养
3. 精神享受
4. 寿元

### 福德宫星曜组合

**紫微星**：福泽深厚，有品味

**天同星**：享乐型，注重生活品质

**太阴星**：暗福，内心丰富

**贪狼星**：欲望强，享受复杂

### 福德宫与寿元

福德宫为寿元宫之一：
- 福德宫强且安稳：寿长
- 福德宫动荡不安：寿有折损

## 父母宫

父母宫反映父母关系，也代表师长和学历。

### 父母宫的核心含义

1. 父母缘分和关系
2. 遗传因素
3. 学历学业
4. 师长久敬

### 父母宫星曜分析

**紫微星**：父母地位高，有助力

**太阳星**：父亲缘强，光明磊落

**太阴星**：母亲缘强，感情深

**天机星**：父母变动，学历变化

### 父母宫与学业

父母宫也代表学业：
- 吉星多：学历高
- 煞星多：学业有阻
- 化科：学业顺利

## 宫位之间的关系

### 三合关系

命宫、财帛宫、官禄宫为三合格：
- 三宫皆吉：人生顺遂
- 三宫皆凶：人生困顿
- 三宫有吉有凶：起伏人生

### 对冲关系

对宫关系：命宫对迁移宫，兄弟宫对仆役宫，夫妻宫对官禄宫等。对宫可补充或对冲本宫。

### 夹宫关系

命宫被兄弟宫和父母宫夹，夹的力量可加强或减弱命宫。

## 宫位与星曜的配合

### 宫主星

每个宫位都有自己的主星，主星强则宫位强。

### 星曜在宫位

同一星曜在不同宫位含义不同：
- 贪狼在命宫：欲望强
- 贪狼在财帛宫：财运多变动
- 贪狼在子女宫：桃花重

### 宫位四化

宫位天干产生四化：
- 化禄入某宫：某方面运势增强
- 化忌入某宫：某方面问题出现
