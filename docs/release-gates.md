# Release Gates

A Codex-generated upgrade is not releasable unless all gates pass.

## Gate 1: Product integrity
- critical routes render
- no missing major sections
- premium upsell still present

## Gate 2: Trust & privacy
- disclaimers present
- share pages hide sensitive birth fields by default

## Gate 3: Code quality
- build passes
- lint passes
- typecheck passes
- tests pass

## Gate 4: Design consistency
- no mirrored bilingual content on marketing pages
- shared CTA language remains consistent
- chart styles remain in one family

## Gate 5: Reviewability
- PR summary generated
- changed files understandable
- follow-up items listed
