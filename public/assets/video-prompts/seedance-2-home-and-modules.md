# TianJi Seedance 2.0 中文视频提示词包

使用方式：每次只复制一个模块的【Seedance 提示词】到 Seedance 2.0。每段提示词都已压缩为 2000 字以内，并且自带风格、镜头、运动和禁止项。

重要：Seedance 上传图片时，请使用“本地首帧/尾帧”对应的 Windows 文件。网页路径只用于视频导出后回填到项目。双图模式下，首帧和尾帧都上传同一张本地图片。

统一画面标准：深空黑、克制紫雾、古金色边光、少量青色地平线 rim light、一个清晰主体、左侧保留文字叠加空间。不要把任何文字、logo、按钮、价格、中文、英文、UI 或水印生成进视频。

## 输出规范

1. Hero 视频：MP4，H.264，6 秒，1080p，文件名严格匹配上方输出路径。
2. 卡片视频：MP4，H.264，3 秒，768p 或更小，文件名严格匹配上方输出路径。
3. 只替换 `public/assets/**` 下的视频文件，不改 API、Stripe、Supabase、auth、路由或请求响应格式。
4. 视频落盘后建议运行：
   - `npm run test -- src/__tests__/landing-design-contract.test.ts`
   - `npm run build`
   - `npm run audit:routes`
   - `npm run audit:share`
