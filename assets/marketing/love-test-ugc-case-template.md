# Love-Test UGC Case Template

Use this only for real, permissioned user-submitted cases. Do not invent testimonials, numbers, revenue, or outcomes.

## Permission Checklist

```text
User permission received: yes/no
Permission phrase or written approval stored privately by operator: yes/no
All names removed: yes/no
Screenshots anonymized: yes/no
Private answers removed: yes/no
No guaranteed outcome claim: yes/no
No medical/diagnostic language: yes/no
No fake testimonial wording: yes/no
```

## Case Structure

### 1. Context

Use broad, anonymous framing:

```text
A user was in a long ambiguous connection and felt unsure whether to act or wait.
```

Do not include:

```text
names, exact city, workplace, school, phone number, screenshots with identities, birth data, private messages
```

### 2. Love-Test Result

```text
Archetype:
Score range:
One-liner:
Safe next step:
```

### 3. User Reflection

Only quote with permission. Prefer paraphrase:

```text
The user said the result helped them pause before sending another message.
```

### 4. CTA

```text
If you are also stuck in a relationship loop, try the free Love Test first: /love-test
```

### 5. Safety Note

```text
This case is shared with permission and anonymized. TianJi Love is a reflection tool, not a guarantee of relationship outcome.
```
